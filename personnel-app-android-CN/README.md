# 碣石人事管理系统

**版本**: v2.1.6  
**日期**: 2026-03-15  
**状态**: ✅ 完整可构建

---

## 📱 应用简介

碣石镇人事信息管理系统是一款**离线 Android 应用**，用于镇/村两级人事信息管理。

### 核心特性
- ✅ 四级菜单架构（启动页→首页→列表→详情）
- ✅ 统一蓝色渐变设计风格
- ✅ 多身份支持（镇府干部/村两委干部/网格联防员）
- ✅ 驻村领导自动排首位
- ✅ 头像 ZIP 包自动匹配
- ✅ Excel 批量导入
- ✅ 全局搜索（中文/拼音）
- ✅ 数据加密存储

---

## 🎨 界面架构

```
启动页（蓝色渐变 + 导入按钮）
  ↓
一级菜单（搜索框 + 三个主按钮）
  ├─ 镇府干部 → 二级菜单（16 个机构单列）
  ├─ 村两委干部 → 二级菜单（42 个村 3 列网格）
  └─ 网格联防员 → 二级菜单（42 个村 3 列网格）
  
二级菜单
  ↓
三级菜单（人员列表：头像 + 姓名 + 职务）
  ↓
四级菜单（人员详情：圆形头像 + 基本信息 + 信息框）
```

## 🚀 快速开始

### 1. 打开项目
```
Android Studio → File → Open → 选择本文件夹
```

### 2. 等待 Gradle 同步
首次同步需要 5-10 分钟

### 3. 构建 APK
```
Build → Build APK(s)
```

**APK 位置**: `app/build/outputs/apk/debug/app-debug.apk`
